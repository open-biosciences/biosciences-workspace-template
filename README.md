# biosciences-workspace-template

Bootstrap scripts and workspace configuration templates for the Open Biosciences platform.

Part of the [Open Biosciences](https://github.com/open-biosciences) multi-repo platform.

## Status

**Planned.** Templates will be created after the core platform migration (Waves 1-3) is complete. Stable repo structures and conventions must be established before templates can be finalized.

## What Will Be Here

- Bootstrap scripts for initializing development environments
- VS Code workspace configuration templates
- Repo scaffolding templates for new platform repos
- Canonical configuration distribution (shared linting, type checking, and test settings)
